# Light and Shadow

Interactive light and shadow scenes built with Python Tkinter.

This project contains two small visual experiments:

- `a.py`: 2D light and shadow scene
- `b.py`: ray-traced 3D light and shadow scene

## Run

```powershell
python a.py
```

```powershell
python b.py
```

## Controls

### a.py

- Left drag: move the light or aim the beam
- Right drag: move the whole beam
- Arrow keys: move the light
- `WASD`: move the aim point
- `Space`: reset
- `Esc`: quit

### b.py

- Drag the square handle: move the light
- Drag the scene: aim at a 3D surface
- Right drag or mouse wheel: move light depth
- `A/D`: move light left and right on screen
- `W/S`: move light forward and backward
- `Up/Down`: move light height
- `Left/Right`: rotate camera
- `Space`: reset
- `Esc`: quit
